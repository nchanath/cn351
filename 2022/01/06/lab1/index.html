LAB 1 -- วันที่ 10 มกราคม พ.ศ. 2565

ขั้นตอนในการทำแลบ

  1. ดาวน์โหลดและติดตั้งโปรแกรม Virtualbox

  2. ดาวน์โหลดและติดตั้งเครื่องเสมือน Web Security Dojo เวอร์ชั่นล่าสุด (ณ ขณะนี้เป็นเวอร์ชั่น 3.4.1) ตั้งชื่อเครื่องนี้ว่า dojo VM

  3. ดาวน์โหลด iso image ของ Ubuntu จาก Ubuntu.com มาเก็บไว้ที่เครื่อง แล้วทำการติดตั้งเป็นเครื่องเสมือนที่ถูกบริหารจัดการด้วย virtualbox ตั้งชื่อเครื่องนี้ว่า Ubuntu VM

  4. Login เข้าเครื่อง dojo VM แล้วทำ Unix lab ของ UC Berkeley ตามที่ปรากฏใน http://www-inst.eecs.berkeley.edu/~cs61a/fa14/lab/lab00/ เนื่องจากนักศึกษาไม่ใช่นักศึกษาของ UC Berkeley จึงไม่มีบัญชีอยู่ที่นั่น ดังนั้นให้ข้ามสิ่งที่เกี่ยวข้องกับการใช้บัญชีของ UC Berkeley เท่าที่จำเป็น

  5. ดาวน์โหลดและติดตั้ง text editor ที่นักศึกษาต้องการใช้ใน Ubuntu VM และ dojo VM แล้วเรียนรู้วิธีการใช้ให้คล่องแคล่ว โปรแกรม editor ที่นักศึกษาอาจพิจารณาใช้ ได้แก่ Emacs, Visual Studio Code, vi, atom และ sublime text เป็นต้น

  6. สำหรับนักศึกษาที่ใช้เครื่อง Windows นักศึกษาอาจทำการตั้งค่าให้เครื่องของตนสามารถใช้งาน WSL2 ในเครื่อง Windows ที่เป็น host ด้วย (หากนักศึกษาใช้ Windows ที่ต่ำกว่าเวอร์ชั่น 11 ต้องทำการติดตั้งโปรแกรมที่ทำหน้าที่เป็น X Server ด้วย เช่น โปรแกรม X410 ที่นักศึกษาสามารถดาวน์โหลดได้จาก Microsoft Store) การทำเช่นนี้จะทำให้มีความสะดวกในการใช้งานมากขึ้น

  7. ให้นักศึกษาฝึกใช้คำสั่งตามวิดีโอ https://youtu.be/cBokz0LTizk แล้วเขียนรายชื่อคำสั่งที่อยู่ในวิดีโอให้ครบทุกคำสั่ง โดยส่งเป็นการบ้านชื่อ Lab 1 write-up ใน MS Teams ผ่าน OneNote class notebook และให้นักศึกษาอัดวิดีโอแสดงการทำงานตามวิดีโอตัวอย่าง เพื่อส่งเป็นวฺ๊วิดีโอสำหรับ Lab 1 ของตนเอง

  8. ในไฟล์เดียวกันกับข้อ 7. ให้นักศึกษาเขียนบรรยายปัญหาหลัก ๆ ที่พบระหว่างการทำแลบพอสังเขปมา 3-5 ข้อ

ในการส่งการบ้านทุกครั้ง ให้นักศึกษาเขียนชื่อและเลขที่ทะเบียนไว้ที่มุมซ้ายบนของไฟล์ทุกครั้ง และให้นักศึกษาเขียนคำตอบด้วยลายมือของตนเองเท่านั้น ห้ามพิมพ์ ในกรณีที่นักศึกษาไม่มี tablet ให้เขียนคำตอบในกระดาษ ถ่ายรูปแนวตั้ง โฟกัสให้ชัดเจนอ่านง่าย แล้วอัพโหลดรูปภาพใส่ไว้ใน notebook

ข้อมูลที่อาจเป็นประโยชน์

  • วิธีการตั้งค่าและติดตั้ง WSL2 และ Ubuntu ในเครื่อง Windows มีหลายวิธี นักศึกษาอาจลองวิธีที่ง่ายตามที่ปรากฏที่ https://docs.microsoft.com/en-us/windows/nodejs/setup-on-wsl2
