LAB 10 -- วันที่ 21 มีนาคม พ.ศ. 2565

Injection flaws

ในแลบนี้นักศึกษาจะใช้ web security dojo VM

  1. ทำแบบฝึกหัด WebGoat เรื่อง Injection Flaws โดยให้ทำแลบในหัวข้อต่อไปนี้
  • Command Injection
  • Numeric SQL Injection
  • Log Spoofing
  • XPATH Injection
  • String SQL Injection
  • LAB: SQL Injection Stages 1 and 3
  • Blind Numeric SQL Injection
  • Blind String SQL Injection
  1. ให้ทำแบบฝึกหัด dvwa ในหัวข้อ SQL Injection เพียงหัวข้อเดียว โดยให้
  • แสดงผลลัพธ์บน browser และ
  • แสดงการส่ง SQL query ที่ใช้ในการโจมตีในข้อนี้ ผ่าน mysql client ที่ terminal และแสดงผลลัพธ์ที่ได้ว่ามีเนื้อหาที่ตรงกันกับผลลัพธ์ที่แสดงบน browser

ในการทำแลบทั้งสองส่วนให้ใช้

  • burp suite สำหรับดักจับข้อมูลระหว่าง browser และ server และ
  • firefox ในการสำรวจ source code

เท่านั้น ห้ามใช้เครื่องมืออำนวยความสะดวกอื่น ๆ เช่น Tamper Data, Live HTTP headers ฯลฯ

รายงานแลบ

ให้เขียนสรุปสั้น ๆ เพื่ออธิบายในแต่ละข้อถึง

  • สาเหตุที่ทำให้เกิดช่องโหว่ และ
  • แนวคิดในการโจมตีของนักศึกษา (เขียนโดยละเอียดพอประมาณ) และ
  • SQL query (หรือ queries) ที่นักศึกษาคิดว่าถูกส่งไปยังฐานข้อมูลเมื่อนักศึกษาทำการโจมตี และคำอธิบายผลลัพธ์ที่ได้ ว่าเกิดจาก SQL query แต่ละค่าที่ส่งไปได้เพราะอะไร อย่างไร และ
  • แนวคิดในการปิดหรือรับมือกับช่องโหว่ (เขียนโดยละเอียดพอประมาณ)
